# Museum site 

Used: 
- typescript
- react
- scss

Online demo: https://relativelie.github.io/form-app/

## How to build
1. Clone repository
``git clone git@github.com:Relativelie/form-app.git /path/to/feedback``
``cd  /path/to/feedback ``
2. Install dependencies
``npm install ``

## How to start
1. Run project 
``npm start ``
